# Описание проекта

**Заказчик:** кредитный отдел банка.

**Цель исследования:** выяснить, влияет ли семейное положение и количество детей клиента на факт погашения кредита в срок.

**Применение результатов:** данные исследования будут использованы при построении модели кредитного скоринга — системы, оценивающей способность потенциального заёмщика вернуть кредит банку.

## Описание данных

В распоряжении — статистика о платёжеспособности клиентов. Ниже — перечень переменных и их краткое описание:

| Переменная | Описание |
|----------|----------|
| `children` | Количество детей в семье |
| `days_employed` | Общий трудовой стаж в днях |
| `dob_years` | Возраст клиента в годах |
| `education` | Уровень образования клиента |
| `education_id` | Идентификатор уровня образования |
| `family_status` | Семейное положение |
| `family_status_id` | Идентификатор семейного положения |
| `gender` | Пол клиента |
| `income_type` | Тип занятости |
| `debt` | Факт наличия задолженности по возврату кредитов (1 — имел задолженность, 0 — не имел) |
| `total_income` | Ежемесячный доход |
| `purpose` | Цель получения кредита |

## Ключевые исследовательские вопросы

1. Есть ли связь между количеством детей и вероятностью своевременного погашения кредита? Отличаются ли клиенты без детей от клиентов с 1, 2, 3 и более детьми по уровню долговой дисциплины?
2. Влияет ли семейное положение клиента на вероятность возникновения задолженности? Например, отличаются ли показатели у клиентов, состоящих в браке, разведённых, холостых и т. д.?
3. Существуют ли комбинации факторов (например, «многодетный и разведённый», «молодой и без детей»), которые особенно сильно влияют на риск невозврата кредита?

## План анализа данных

### 1. Предобработка данных

* проверка на пропуски и их устранение;
* выявление и обработка аномалий (например, отрицательное количество детей, неправдоподобно высокий стаж);
* приведение типов данных к корректному формату;
* поиск дубликатов и их удаление.

### 2. Исследовательский анализ данных (EDA)

* расчёт базовых статистик по ключевым переменным (`children`, `family_status`, `debt`);
* группировка данных по категориям семейного положения и количеству детей;
* подсчёт доли клиентов с задолженностью в каждой группе.

### 3. Анализ взаимосвязей

* построение сводных таблиц и графиков для визуализации зависимости между семейным положением / количеством детей и наличием задолженности;
* статистическая проверка значимости различий (например, с помощью критерия хи‑квадрат или других методов).

### 4. Формулировка выводов

* ответ на основные исследовательские вопросы;
* рекомендации для модели кредитного скоринга (какие факторы стоит учитывать).
